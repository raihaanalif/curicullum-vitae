Project Host:

Binar Academy Exercise Design

Project Date:

April 2022

Project Description:

Make an application design to help cake shop owner to sell their cake to customer with online method

Project Role:

My role in this project as UI UX Designer

Project Overview

The owner cake shop want to sell their cake but the customer is very lazy to go outside to buy a cake.

Define

The project started with define phase which is the goals is to find the user personas and user journey. Before that, i took a bechmarking first between 2 biggest food order applications in Indonesia which is GrabFood and GoFood and find the lack of 2 application from user interview with the user persona that very suitable with this project. This is the user persona that i choose for this project.

After i got the user persona, i start to interview them and find out their problem about two application that i got to benchmarking. The problem that they mention i record into user journey map. You can see the user journey map here.

Ideation

In this phase, i start to make an ideation from point point that i got from user journey map. With brainstromming i start to collect the idea that can i make an improvement to the cake shop application. This is the idea that i collect from user journey map.

After i got the idea, i start to make flowchart before i make a wireflow. This is the flowchart and wireframe that i make to finish this step.

Prototype

After i finish the wireframe, i start to make a prototype from the point that i got from the pain point on interview with some user. But before i got this prototype, i need to make a design system first to documentation this project and got an easier way to make a prototype. This is it the design system that i made for this project, the font that i used to this project is Google Font Poppins because a lot of online order food like Gofood and GrabFood use the similar or this font. And size of this application font is refers to material design website.
