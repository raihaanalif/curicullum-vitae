Project Role:

Project Overview

This project is to find an applicaton that have a lack of user experience and make it more friendly to use. I choose discord because i frequently use this application and one of my friend sometimes complaining about discord design.

From points that my friends always complaining is in menu navigation bar that very unfriendly because the icon that they use is different than other application navigation bar. Because of that, i also must to change all of application layout to make the navigation bar more friendly and easy to recognize. After i get the point that must be change, i make a design system to make a prototype more easier. Because it is a redesign project, i took a lot of assets from discord guideline website and font awesome website from universal icon pack. and this is the design system.
